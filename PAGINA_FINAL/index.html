Sobre nós

Desenvolvemos esse software para o Projeto Interdiciplinar do curso de DSM. O objetivo da FATECtools é facilitar os estudantes a terem todos os programas
utilizados em um lugar só, também possui conteúdos adicionais para complementar o estudo. Inicialmente o software atende apenas o curso de DSM, porém
futuramente poderá atender outros cursos da FATEC JAHU.
