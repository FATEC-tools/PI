Figma

Descrição

Figma é um editor gráfico de vetor e prototipagem de projetos de design baseado principalmente no navegador web, com ferramentas offline adicionais para aplicações desktop para GNU/Linux, macOS e Windows.

Primeiros passos

1º Passo

Depois de baixar o Figma no site, selecione "FigmaSetup" para instalar na sua maquina.

2º Passo

Depois de baixar o figma, é necessario criar uma conta para poder usar, crie uma ou use um login caso já tenha feito.

3º Passo

Para começar um projeto clique em "design file", caso tenha duvidas de como usar as ferramentas assista ao vídeo abaixo ou siga o tutorial disponível no próprio aplicativo.
